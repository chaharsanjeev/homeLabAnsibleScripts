
1. Copy two files from GitHub into Proxmox PVE server and copy at PVE server folder "/etc/"
    - vzdump-hook-script.pl  
    - vzdump-hook-sub-insert-mysql.sh

  So now PVE server under "/root/etc" folder will have the above two files

2. provide execute access to both these files
   - chmod 777 vzdump-hook-script.pl
   - chmod 777 vzdump-hook-sub-insert-mysql.sh

3. Now edit file "nano /etc/vzdump.conf" at PVE server and add below line into this file "/etc/vzdump.conf"

     script: /etc/vzdump-hook-script.pl

4. restart PVE server and test it
